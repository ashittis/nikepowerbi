# 👟 Nike Sales Dashboard - Power BI Report

This Power BI dashboard provides a comprehensive overview of Nike's sales performance across various metrics such as region, product category, revenue, profit, and customer behavior.

## 📊 Key Features

- 📈 **Sales Trends Over Time**  
  Track revenue and profit trends monthly, quarterly, or yearly.

- 🌍 **Geographical Sales Distribution**  
  Visualize performance across different countries or regions.

- 🏷️ **Product Category Breakdown**  
  See top-performing product categories and SKUs.

- 💵 **Revenue and Profit Analysis**  
  Understand revenue contributions and profit margins.

- 👤 **Customer Insights**  
  Analyze customer segments, purchase behavior, and retention.

- 📦 **Inventory vs. Sales**  
  Compare stock availability against actual sales.

- 👟 **Shoe-wise Analysis**  
  Drill down into individual shoe models to track their sales, profit margins, and stock availability. Identify bestsellers and underperformers using product-level visualizations.

## 🛠️ Built With

- **Power BI Desktop (.pbit template)**
- Custom visuals (e.g., Card (New), Map, KPI, Slicer)
- DAX Measures and Calculated Columns

## ✅ Usage Instructions

1. Open `Nike_Sales_DashBoard.pbit` in Power BI Desktop.
2. Connect your dataset if prompted (e.g., CSV, Excel, database).
3. Explore the report tabs to analyze different sales KPIs.
4. Use filters/slicers to customize views by:

   * Region
   * Time period
   * Product category
   * Customer segment
